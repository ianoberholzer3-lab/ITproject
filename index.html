TASK 1 A:

Leerlinge moet n toeganklike program he om hulle werk na te gaan en seker te maak all hulle antwoorde is reg terwyl hulle hul huiswerk nagaan en doen


Ek gaan n simple sakrekenaar skryf wat leerlinge toelaat om hulle basic math the kan nagaan


Ek gaan n title skerm maak wat sal maak laat die gebruiker n knop kan druk wat jou of na die sakrekenaar self toe laat gaan of bv n instellings tab

Wat bv n donker modus wat beter sal werk op die gebruiker se oe

TASK 1 B:

As n leerling ek will graag n sakrekenaar he sodat ek beter kan vaar in my wiskunder deur om myself na te gaan


As n persoon met sensitiewe oe ek will graag n sakrekenaar he wat n donker modus opsie het sodat ek die GUI beter kan sien

TASK 2:

Die stelsel sal n correcte antwoord vir die som kan gee (wat die gebruiker ingetik het) as jy die = knoppie druk


Die stelsel sal moet n error boodskap gee wander die gebruiker n wiskundige incorrecte som intik(divide by 0)


Die stelsel sal n setting screen opsie moet kan he vir "donker modus"


Die stelsel se agtergrond sal donkerder gaan deur om ale helder GUI te laat verdwyn

MEER INFORMASIE:

Hoekom is die program geskryf: Die program is geskryf asgevolg van n IT pat wat die leerder(skrywer) vra om n program te skryf wat die gebruiker iets kan leer of kan help met hulle studies

Op watse stelsel sagteware kan hierdie program hardloop: Die program kan op linux , Winows 10-11 en MacOS hardloop

Wat is die requirments vir hierdie program: Hierdie program het nie engige vaste requirments om te kan hardloop nie

Hoekom het hierdie program n titel skerm: Hierdie program het n titel skerm omdat hy geskryf was om verskeie take kan verug hoekom ek so se is as die gebruiker so wou meer informasie van die program weet kan hy net op die "info" knoppie druk of as hy van die program se settings verander kan hy net op die "settings" knoppie druk maaar as die gebruiker net die sakrekenaar will gebruik kan hy net op die "calculater" knoppie druk ek het dit asook so gedoen om dit meer eenvoudig te laat voel en meer netjies te laat lyk

Kan hierdie program op n foon hardloop: Nee hierdie program kan nie op n foon hardloop nie hy is net gebruik vir rekenaars of laptops
